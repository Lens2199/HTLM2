# HTLM2

## Project Overview
In this project, you are required to create a simple HTML-only website with multiple pages. The website should have the following pages:
- Homepage
- Projects
- Articles
- Contact

The website should have a navigation bar that is present on all pages and links to all the pages.

## Project Goals
- Learn how to create multiple pages in a website.
- Structure a website using HTML in a semantic way.
- Structure in a way that you can easily add styles later.
- Add SEO meta tags to the website.

## Instructions
1. Create the structure of the website using HTML.
2. Ensure the navigation bar is present on all pages and links to all the pages.
3. No styling is required for this project, focus on the HTML structure.
4. Add appropriate SEO meta tags to each page.

## Pages
1. **Homepage**: Introduction to the website.
2. **Projects**: List of projects.
3. **Articles**: Collection of articles.
4. **Contact**: Contact information and form.

## SEO Meta Tags
Ensure each page has the following meta tags:
- `<meta charset="UTF-8">`
- `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
- `<meta name="description" content="Description of the page">`
- `<meta name="keywords" content="HTML, website, multiple pages">`

## Future Enhancements
- Add CSS styles to enhance the visual presentation.
- Implement JavaScript for interactive features.
